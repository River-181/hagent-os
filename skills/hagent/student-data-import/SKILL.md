# 학생 데이터 가져오기

## 목적

기존 학원 운영 자료를 HagentOS 구조에 맞게 빠르게 정리합니다.

## 입력 예시

- 학생 이름 / 학년 / 반
- 학부모 연락처
- 수강 과목
- 최근 출결 또는 상담 메모

## 정규화 규칙

- 이름과 연락처는 필수로 확인합니다.
- 반/과목이 혼재되면 먼저 구분 테이블을 만듭니다.
- 상담 메모는 원문을 보존하고 요약본을 따로 둡니다.
